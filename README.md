## Description:

Project is about finding difference between files using cli tools


## install:

make install

## build:

make build

## package install:

make package-install

## usage:

gendiff [options] <file_path1> <file_path2>

**How to run script with instructions:**

`gendiff -h`


**To remove project just run `rm -rf .`**
